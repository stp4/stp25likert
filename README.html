Likert-Grafiken

Likert-Skalen sind Fragenbatterien, die mit vorgegebenen mehrstufigen Antwortskalen beantwortet werden. Die dazugehrigen Fragen (Items) sind als strikt positive oder negative Aussagen formuliert.

Die Auswertung erfolgt bei rein deskriptiven Fragestellungen durch Auszhlung und Berechnung der Anteilswerte. Wenn die Stichprobe gro genug ist, kann eine Likertskala auch als metrische Skala angesehen werden und es knnen dann alle Kennzahlen wie z.B. Mittelwerte und Korrelationen berechnet werden.

Klassiker Plot mit der HH Library

Die Funktion likertplot() ist ein Workaround fr die Funktion HH:::plot.likert.formula. Dabei kann ein Objekt, das mit der Funktion Tbll_likert() erzeugt wurde, mit allen notwendigen Parametern direkt bergeben werden. Mit HH lassen sich die Referenzlinien verschieben.

cowplot::plot_grid(
  likertplot(dat_l,
             auto.key = list(columns = 5)),
  likertplot(
    Item ~ . | Geschlecht,
    data = dat_lg,
    auto.key = list(columns = 5)
  ),
  ncol = 2,
rel_widths = c(2,3)
)

Hndich mit ggplot

dat <- attr(dat_l, "plot")$results
save_lvl <- names(dat)
names(dat) <- c("Item", "l1", "l2", "l3", "l4", "l5")

dat <-   Long(. ~ Item , dat, value = "Freq")
levels(dat$variable) <- save_lvl[-1]

  ggplot(dat) +
  aes(
    x = Item,
    fill = variable ,
    weight = Freq,
    by = Item
  ) +
  geom_bar(position = "fill") +
  geom_text(
     aes(label = scales::percent(after_stat(prop), accuracy = 1)),
    stat = "prop", position = position_fill(.5)) +
    coord_flip()
